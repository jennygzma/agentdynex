**Front End** <br />
Make sure npm, react, and mui are installed<br />

Commands: <br />
`cd frontend`<br />
`npm start`<br />

**BackEnd**<br />
in another terminal: <br/>

Commands: <br/>
`cd backend`
`conda env create -n UIDesignProto -f environment.yml`<br />
pip install as necessary<br />
`conda activate UIDesignProto`<br />
`python server.py` <br />
